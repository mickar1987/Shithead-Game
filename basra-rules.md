# Basra Rules Summary

## Setup
- 52 cards, 2 or 4 players
- Deal 4 cards each, 4 face-up to table
- If Jack appears in initial 4 table cards → return to deck, redraw
- 7♦ in initial table cards → also return and redraw (Jawaker rule)

## Gameplay
- Play 1 card per turn
- Card can CAPTURE if:
  - Same rank as any table card (pair)
  - Sum of multiple table cards = rank of played card (combine)
  - Both pairing AND combining in same play
  - Jack → captures ALL cards on table
  - 7♦ → captures ALL cards on table
  - Q captures Q only, K captures K only (no numeric value)
- If no capture → card stays on table
- When hand empty → deal 4 more each (no new table cards)
- Round ends when deck exhausted

## Basra (bonus)
- Clear ALL table cards with a single numeric card → BASRA = 10 pts
- 7♦ clears all + table total ≤ 10 + no face cards → BASRA = 10 pts  
- Jack clears all → NOT a basra
- After last card played → last capturer takes remaining table cards (no basra)

## Scoring (per round)
- Most cards (27+): 30 pts (tie = 0, carried to next round)
- Each Ace: 1 pt
- Each Jack: 1 pt
- 2♣: 2 pts
- 10♦: 3 pts
- Each Basra: 10 pts

## Game end
- First to reach 101 pts wins
